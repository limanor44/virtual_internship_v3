**Контекст задачи (краткое описание) **
Узнать какие действия выполняет команда build в Gradle


**Что было сделано и как? (краткое описание)**
Справа нажали на значек Gradle в нем выбрали Task->build  и выбрали команду build, которая запустила сборку всего проекта
Если бы требовалось сделать сборку только модуля то internship -> Tasks -> build -> build
**Зачем это было сделано? Чему научился в процессе выполнения задания?**
Научился делать сборки с помощью програмных и консольных инструментов.
При выполняет команда build в Gradle происходит сборка проека:
1.Проводятся тесты
2.Компилируется исходный код проекта
3. Проверка качества кода
4.Подготовка к развертыванию
**Сколько потратил времени на выполнение задачи?**
5 мин

**Заметки, идеи, предложения в свободной форме**
Для лучшего запоминая консольной команды сборки Gradle "gradlew.bat", запомнил что идет идет наименование сборщика
и последняя буква w - work.
